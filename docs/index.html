ProjectSite

Static site generator for GitHub Pages

Generates self-contained Dracula dark + Alucard light themed HTML project pages from TOML data plus a Jinja2 template. Batch-generates all projects from one command.

View on GitHub

Features

TOML-Driven

Write project data in clean TOML. No HTML editing needed. Each project gets one .toml file with all its content.

Jinja2 Template

Single template with Dracula theme, JetBrains Mono font, sidebar navigation, and responsive layout. Update the look of all projects by editing one file.

Batch Mode

Generate all project pages at once with --batch. Each .toml becomes docs//index.html — ready for GitHub Pages.

Per-Project CI

Each project repo runs the generator in its own GitHub Actions workflow. The .toml is the single source of truth.

CI Workflow

name: Deploy project page

permissions:
  contents: write

on:
  push:
    branches: [main]
    paths:
      - "project.toml"

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: "3.11"
      - name: Checkout generator
        uses: actions/checkout@v4
        with:
          repository: yuzu-octopus/ProjectSite
          path: generator
          persist-credentials: false
      - name: Install system deps
        run: sudo apt-get update && sudo apt-get install -y libcairo2-dev
      - name: Generate page
        run: |
          pip install jinja2 pillow cairosvg
          python generator/genpage.py --input project.toml --output docs/index.html
      - name: Commit generated page
        run: |
          git config user.name "github-actions[bot]"
          git config user.email "41898282+github-actions[bot]@users.noreply.github.com"
          git add docs/
          git diff --cached --quiet || git commit -m "ci: regenerate project page [skip ci]"
          git push https://${{ github.actor }}:${{ secrets.GITHUB_TOKEN }}@github.com/${{ github.repository }}.git HEAD:${{ github.ref_name }}

Usage

# Single project
uv run python genpage.py --input project.toml --output docs/index.html

# Batch mode — all .toml files in a directory
uv run python genpage.py --input projects/ --output docs/ --batch

def fibonacci(n: int) -> list[int]:
    """Generate first n Fibonacci numbers."""
    if n <= 0:
        return []
    fib = [0, 1]
    for _ in range(2, n):
        fib.append(fib[-1] + fib[-2])
    return fib[:n]

# Example usage
result = fibonacci(10)
print(f"First 10 Fibonacci: {result}")
